# Data_Mining
**미국 대통령선거 트럼프 지지자 예측**
- 소개 : 미국 대통령선거 트럼프 지지자 예측하기
- 데이터 : 2018년 미국 일반사회조사 자료
- 사용 모형 
  * glm
  * glmStepAIC
  * glmnet
  * nnet
  * svmRadial
  * rpart
  * ranger
  * gbm
  * xgb

- 결과
  * 최종모형으로 Ranger을 선택
  * TS에 대한 예측값이 가장 높은 5명(교회참여정도, 흑인, 나이, 부친 사회경제지표, 가구소득)
과 가장 낮은 5명(별거, 황인, 미망인, 이혼, 미혼)을 예측
